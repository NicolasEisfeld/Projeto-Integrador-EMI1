Análise

Gamificação do Pré-IFRS Erechim


Introdução

Tivemos dois bancos de dados para analisar durante os 2 primeiros trimestres

  • - Pesquisa feita nos alunos do Instituto em relação ao Pré-IFRS.
  • - Acesso aos dados referente aos alunos ingressantes de 2024 para analisarmos as escolas dos alunos do processo seletivo.

    • Através disso, descobrimos que o Pré-IFRS não é uma ferramenta em que a maioria dos alunos utilizaram, ou até mesmo ter conhecido durante o processo seletivo. Para podermos fundamentar uma ferramente virtual para os estudos preparatórios para o ingresso do Instituto, primeiramente devemos trabalhar na divulgação e na elaboração de um material de qualidade para o Pré-IFRS Campus Erechim.

    Nesse aspecto, é importante divulgarmos nosso projeto presencialmente em escolas públicas que tem esta possibilidade, para facilitar o acesso as informações referentes ao próprio Instituto e o curso preparatório para o ingresso.
