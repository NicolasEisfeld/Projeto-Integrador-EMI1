Projeto Integrador

Gamificação do Pré-IFRS Erechim


Este site foi feito para uma avaliação trimestral da matéria de Desenvolvimento Web I, para divulgar e apresentar os demais aspectos do Projeto Integrador I do aluno. O Projeto Integrador Pré-IFRS Erechim tem como princípio, colaborar e auxiliar de forma prática e intuitiva para os estudantes de escolas públicas do Ensino Fundamental II que não teve uma oportunidade se preparar para o processo seletivo do IFRS Campus Erechim.


Nesse projeto vai ser implementado em uma solução técnica através de uma ferramenta útil e inovadora, que vai ser trabalhada e desenvolvida em 3 anos, iniciando somente com pesquisa e estudo da metodologia posteriormente aplicada, e finalizando com uma aplicação em website de fácil acesso para os próximos estudantes do processo seletivo. Todo este conteúdo do Projeto estará sendo abordado em tópicos, em que pode ser acessado através da barra de navegação horizontal.
